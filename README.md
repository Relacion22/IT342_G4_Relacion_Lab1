AccessHub - User Authentication SystemProject DescriptionAccessHub is a secure authentication system designed to bridge user access across web and mobile platforms. The system focuses on secure user onboarding and session management, ensuring that sensitive data like passwords are never stored in plain text and that system resources are protected by industry-standard authentication filters.Technologies UsedBackend: Java 19, Spring Boot 4.0.2, Spring Data JPA, Spring Security (BCrypt & JWT).Database: MySQL (Persistence) and H2 (Development/Testing).Web App: React.js, Axios, React Router.Mobile App: React Native, Expo.üöÄ Steps to Run BackendDatabase Configuration:Create a MySQL database: CREATE DATABASE accesshub_db;.Open src/main/resources/application.properties and update:Propertiesspring.datasource.url=jdbc:mysql://localhost:3306/accesshub_db
